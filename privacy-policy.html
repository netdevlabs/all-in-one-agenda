Privacy Policy:
  • All-in-One Agenda Widget gathers information from your android phone's calendar, contacts and/or other sources depending on additional addons that are/can be enabled as in-app features (like Facebook integration).
  • In case of location-dependent features are enabled, like weather forcast, All-in-One Agenda uses your phone's approximate geographical location.
  • All these data access is enabled by user permission at the moment of installation.
  • Facebook data access is enabled by user permission on logging-in the application to access Facebook's profile. Data requested from Facebook is user's Facebook events, friend list information (like birthdates) and related data.
  • All-in-One Agenda has a functionality to send debug reports to us (the developers) to be able to investigate and fix potential issues. These debug reports are sent only on user consent and contain only technical information, but may also indirectly contain some hints on user data as well, but not the data itself.
  • No user's data is ever shared with any 3rd party, except for services that user accepted to use within the application (like Facebook integration, geo-location for weather forecasts or alike).
